# 🤖 Crypto Promo Bot - Бот для отслеживания промоакций криптобирж

Универсальный Telegram бот для автоматического отслеживания промоакций криптобирж через их API.

## 🚀 Быстрый старт

### 1. Установка зависимостей
\\\powershell
# Активируем виртуальное окружение
python -m venv venv
venv\Scripts\activate

# Устанавливаем зависимости
pip install -r requirements.txt
\\\

### 2. Настройка бота
1. Получите токен бота у @BotFather в Telegram
2. Замените \your_actual_bot_token_here\ в файле \.env\ на ваш токен

### 3. Запуск бота
\\\powershell
python main.py
\\\

## 📋 Функционал

### ✅ Реализовано:
- ✅ Универсальный парсер для ЛЮБОГО JSON API
- ✅ Добавление/удаление API ссылок
- ✅ Индивидуальные интервалы проверки
- ✅ Тестирование API ссылок
- ✅ Уведомления о новых промоакциях
- ✅ Автоматическое определение уникальных ID промо
- ✅ Игнорирование изменений существующих промо

### 🎯 Основные команды:
- \/start\ - запуск бота
- \/help\ - справка
- \📊 Список ссылок\ - управление API
- \➕ Добавить ссылку\ - добавить новую биржу
- \🔄 Проверить все\ - ручная проверка
- \⚙️ Настройки\ - настройки бота

## ❗ Важно

- Бот работает 24/7 при запущенном \main.py\
- Все данные хранятся в SQLite базе (\data/database.db\)
- Логи сохраняются в папке \logs/\
- Для остановки бота нажмите \Ctrl+C\ в терминале

## 🆘 Поддержка

При возникновении проблем:
1. Проверьте что токен бота указан правильно в \.env\
2. Убедитесь что все зависимости установлены
3. Проверьте логи в папке \logs/\
